python的四种数据结构

列表(list)

首先我们从列表开始,深入讲解每一中数据结构,列表具有显著的特征是:

列表中的每一个元素都是可变的;
列表中的元素是有序的,也就是说每一个元素都有一个位置;
列表可以容纳python中任何对象。
列表的元素是可变的，这意味着我们可以在列表中添加、删除和修改元素。
列表可以容纳任何对象

列表值接受用位置进行索引,单如果数据量很大的话,肯定会记不住什么元素在什么位置,那么有没有一种数据类型可以用人类的方式进行索引哪? 其实就是字典

字段(Dictionary)

编程世界中其实有很多概念都基于现实生活的原型,字典折后在那个数据结构的特征也正如现实世界中的字典一样,使用名称-内容进行数据的构造,在pyhon中分别对象这键(key)-值(value),习惯上称之为键值对。
字典的特征如下:

字典中数据必须是以键值对的形式出现;
逻辑上讲,键是不能重复的,而值是可以重复的;
字典中的(key)是不可变的,也就是无法修改;而值(value)是可变,可以修改的,可以是任何对象

数据结构的一些技巧

推导式现在我们来看数据结构中的推导式（Listcomprehension），
也许你还看到过它的另一种名称叫做列表的解析式，在这里你只需要知道这两个说的其实是一个东西就可以了。
现在我有10个元素要装进列表中，
普通的写法是这样的：
a=[]
for i inrange(1,11):
a.append(i)

下面换成列表解析的方式来写：
b=[i for i in range(1,11)]
列表解析式不仅非常方便，并且在执行效率上要远远胜过前者，
